# Assets

Place lesson images and audio files in these folders:

## Images (`/assets/images/`)
- `alphabet/apple.webp`, `alphabet/ball.webp`, ... (one per lesson)
- `numbers/zero.webp`, `numbers/one.webp`, ...
- `shapes/circle.webp`, `shapes/square.webp`, ...
- `colors/red.webp`, `colors/blue.webp`, ...
- `animals/dog.webp`, `animals/cat.webp`, ...
- `body/head.webp`, `body/eyes.webp`, ...
- `manners/please.webp`, `manners/thankyou.webp`, ...
- `household/chair.webp`, `household/table.webp`, ...
- `food/grains.webp`, `food/vegetables.webp`, ...

## SVG Trace Templates (`/assets/images/traces/`)
- `traces/letter-A.svg` ... `traces/letter-Z.svg`
- `traces/number-0.svg` ... `traces/number-9.svg`
- `traces/shape-circle.svg`, `traces/shape-square.svg`, ...

## Audio (`/assets/sounds/words/`)
- `alphabet/apple.mp3`, `alphabet/ball.mp3`, ...

**Note:** The app uses the Web Speech API as a fallback when audio files are missing,
so the app works fully even before assets are added.
