    1. Create `server/src/lib/sm2.js` as a CJS module exporting four functions:
       - `getThresholds(ageGroup)` — returns `{ easy, medLow }` object. Lookup table:
         `'3-4': { easy: 70, medLow: 50 }`, `'5-6': { easy: 75, medLow: 60 }`, `'7-8': { easy: 80, medLow: 65 }`.
         Default (null/unrecognized): `{ easy: 75, medLow: 60 }`.
       - `classifyAccuracy(accuracyPct, ageGroup)` — returns `'easy'|'medium'|'hard'` by comparing `accuracyPct` (0-100 float) against thresholds.
         `>= easy` -> 'easy', `>= medLow` -> 'medium', else -> 'hard'.
       - `getHardThreshold(ageGroup)` — returns the `medLow` value (the numeric cutoff below which a score is "hard").
       - `applySM2({ interval, easeFactor, reviewCount }, scorePercent)` — implements the exact SM-2 formula from CONTEXT.md:
         ```
         q = (scorePercent / 100) * 5
         if q >= 3:
           if reviewCount === 0: newInterval = 1
           elif reviewCount === 1: newInterval = 6
           else: newInterval = Math.round(interval * easeFactor)
           newEaseFactor = Math.max(1.3, easeFactor + 0.1 - (5 - q) * (0.08 + (5 - q) * 0.02))
         else:
           newInterval = 1
           // easeFactor unchanged on failure
         dueDate = new Date(); dueDate.setDate(dueDate.getDate() + newInterval)
         return { interval: newInterval, easeFactor: newEaseFactor, reviewCount: reviewCount + 1, dueDate, lastReviewedAt: new Date() }
         ```
       Module exports: `module.exports = { applySM2, classifyAccuracy, getHardThreshold, getThresholds };`
