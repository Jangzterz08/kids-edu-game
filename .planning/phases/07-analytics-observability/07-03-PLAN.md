**Step 1 — Prisma schema:**
Add to `server/prisma/schema.prisma` after the last model:
```prisma
model Session {
  id              String     @id @default(uuid())
  kidId           String
  kid             KidProfile @relation(fields: [kidId], references: [id], onDelete: Cascade)
  startedAt       DateTime   @default(now())
  endedAt         DateTime?
  lastHeartbeatAt DateTime   @default(now())

  @@index([kidId, startedAt])
}
```
Add `sessions Session[]` to the KidProfile model's relation list (after `classrooms ClassroomStudent[]`).

Run `cd server && npx prisma migrate dev --name add_sessions`.

**Step 2 — sessions.js route:**
Create `server/src/routes/sessions.js`:
- `POST /heartbeat` — protected by `requireAuth` (kid auth via `req.kidProfile`).
  - Body: `{ sessionId? }` (optional — null means create new session).
  - If `sessionId` is null/undefined: create new Session with `kidId = req.kidProfile.id`, return `{ sessionId: session.id }`.
  - If `sessionId` is provided: update Session where `id = sessionId AND kidId = req.kidProfile.id`, set `lastHeartbeatAt = new Date()`. Return `{ sessionId }`.
  - If `sessionId` provided but not found: create new Session (client lost its reference).
- `POST /end` — protected by `requireAuth`.
  - Body: `{ sessionId }`.
  - Update Session where `id = sessionId AND kidId = req.kidProfile.id`, set `endedAt = new Date(), lastHeartbeatAt = new Date()`. Return `{ ok: true }`.

**Step 3 — analytics.js route:**
Create `server/src/routes/analytics.js`:
- `GET /parent/:childId` — protected by `requireAuth` (parent auth via `req.user`).
  - Query param: `period` (default `7d`).
  - **Privacy check:** Fetch `prisma.kidProfile.findFirst({ where: { id: childId, parentId: req.user.id } })`. If null, return 403 `{ error: 'Access denied' }`.
  - **Daily minutes:** Fetch sessions for the kid in the last N days. Use `prisma.session.findMany({ where: { kidId: childId, startedAt: { gte: cutoffDate } } })`. In JS, compute duration per session as `(endedAt || lastHeartbeatAt) - startedAt` in minutes. Group by date (startedAt date string). Fill missing dates with 0. Return array of `{ date: 'YYYY-MM-DD', minutes: N }` for each day in the period.
  - **Module stars:** Fetch `prisma.lessonProgress.findMany({ where: { kidId: childId, updatedAt: { gte: cutoffDate } }, include: { lesson: { include: { module: true } } } })`. In JS, group by `lesson.module.slug`, compute `avgStars = sum(starsEarned) / count`. Return array of `{ slug, name, avgStars }`.
  - Response shape: `{ dailyMinutes: [...], moduleStars: [...] }`.

**Step 4 — Wire routes in index.js:**
In `server/src/index.js`, add after existing route registrations:

const sessionsRoutes = require('./routes/sessions');
app.use('/api/sessions', sessionsRoutes);
