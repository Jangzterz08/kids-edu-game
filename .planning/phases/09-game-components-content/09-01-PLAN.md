**Imports:**
```jsx
import { useState, useEffect } from 'react';
import { speakWord } from '../../lib/sound';
```

**Component signature:**
```jsx
export default function TrueFalseGame({ lessons, onComplete })
```

**State:**
- `currentIdx` (number, starts at 0) — index into lessons array
- `answered` (null | 'correct' | 'wrong') — tracks answer state for feedback
- `disabled` (boolean) — prevents double-tap after answering

**Current lesson:** `const lesson = lessons[currentIdx]`
- If no lessons: return centered text "No activities yet for your age group. More coming soon!"

**Auto-speak on mount/lesson change:**
```jsx
useEffect(() => { if (lesson?.claim) speakWord(lesson.claim); }, [currentIdx]);
```

**handleAnswer(userChoice):**
1. If `disabled` return early
2. Set `disabled = true`
3. Compare `userChoice` (boolean) with `lesson.correct` (boolean)
4. If match: set `answered = 'correct'`, score = 100
5. If no match: set `answered = 'wrong'`, score = 0
6. After 1500ms: if more lessons, advance `currentIdx` and reset `answered`/`disabled`; else call `setTimeout(() => onComplete(score), 700)`

**For single-lesson TrueFalse (common case):** The component calls onComplete after the first answer. For multi-lesson: iterates through lessons, computing average score across all answers.

**Render layout (stacked vertically, centered, max-width 360px):**
1. **Claim area:**
   - If `lesson.claimEmoji` exists (ages 4-6): show emoji at `fontSize: '64px'` centered, no text
   - Else (ages 6-8): show `lesson.claim` text at `fontSize: 'var(--font-md)'`, `fontWeight: 600`
2. **Replay button:** `<button onClick={() => speakWord(lesson.claim)} aria-label="Replay">` with text content `🔊` at `fontSize: '28px'`, styled as a small circle button
3. **True button:** Full-width, `min-height: 72px` (var(--btn-height)), `borderRadius: 36px` (var(--btn-radius)), label "True ✓"
   - Neutral: `background: 'var(--btn-blue-base)'`, `boxShadow: '0 8px 0 var(--btn-blue-shade)'`
   - If answered correct AND lesson.correct===true: `background: 'var(--btn-green-base)'`, `animation: 'pop-correct 0.55s ease'`
   - If answered wrong AND lesson.correct===true: `background: 'var(--btn-green-base)'` (highlight correct)
   - If answered wrong AND lesson.correct===false: `background: 'var(--accent-red)'`, `animation: 'shake 0.5s ease'`
4. **False button:** Same as True button but label "False ✗"
   - Mirror logic: show green on the correct answer, red+shake on the wrong answer
5. Both buttons get `pointerEvents: 'none'` when `disabled` is true

**Styles object** at bottom of file:
- `container`: `{ display: 'flex', flexDirection: 'column', alignItems: 'center', padding: 'var(--space-lg)', maxWidth: 360, margin: '0 auto', gap: 'var(--space-md)' }`
