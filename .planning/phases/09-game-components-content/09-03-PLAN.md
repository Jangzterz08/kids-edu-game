**Imports:**

import { useState, useMemo } from 'react';

**shuffle function** (Fisher-Yates, same as MatchingGame):
```jsx
function shuffle(arr) {
  const a = [...arr];
  for (let i = a.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [a[i], a[j]] = [a[j], a[i]];
  }
  return a;
}
```

**Component signature:**
```jsx
export default function SortGame({ lessons, onComplete })
```

**Setup:**
```jsx
const lesson = lessons[0]; // Sort uses first lesson
const correctOrder = lesson.items; // items array IS the correct order
const shuffled = useMemo(() => shuffle([...lesson.items]), []);
```

**State:**
- `order` (array) — current arrangement of items, initialized from `shuffled`
- `selectedIdx` (number | null) — currently selected item index
- `checked` (boolean) — whether Check has been tapped
- `results` (array | null) — per-item boolean correct/wrong after Check

**handleItemTap(idx):**
1. If `checked` return (no interaction after reveal)
2. If `selectedIdx === null`: `setSelectedIdx(idx)` — Phase 1: select
3. If `selectedIdx === idx`: `setSelectedIdx(null)` — deselect same item
4. Else: swap `order[selectedIdx]` and `order[idx]`, `setSelectedIdx(null)` — Phase 2: swap

**handleCheck():**
1. Compute `results` array: for each item in `order`, compare its `emoji` to `correctOrder[i].emoji` — `true` if match, `false` if not
2. `setResults(results)`, `setChecked(true)`
3. Compute `correctCount` = number of `true` in results
4. `score = Math.round((correctCount / correctOrder.length) * 100)`
5. After 1500ms display, call `setTimeout(() => onComplete(score), 700)`

**Render layout:**
```jsx
<div style={styles.container}>
  <h2 style={styles.prompt}>{lesson.prompt}</h2>
  <div style={styles.itemRow}>
    {order.map((item, i) => (
      <button
        key={item.emoji + i}
        onClick={() => handleItemTap(i)}
        style={{
          ...styles.item,
          ...(selectedIdx === i ? styles.itemSelected : {}),
          ...(checked && results[i] ? styles.itemCorrect : {}),
          ...(checked && !results[i] ? styles.itemWrong : {}),
          animation: checked && !results[i] ? 'shake 0.5s ease' : 'none',
        }}
      >
        <span style={{ fontSize: item.renderSize + 'px', lineHeight: 1 }}>{item.emoji}</span>
      </button>
    ))}
  </div>
  {!checked && (
    <button onClick={handleCheck} className="kid-btn" style={styles.checkBtn}>
      Check ✓
    </button>
  )}
  {checked && (
    <div style={{ ...styles.resultText, animation: 'slide-up 0.3s ease' }}>
      {results.every(r => r)
        ? <span style={styles.perfect}>Perfect! 🎉</span>
        : <span>{results.filter(r => r).length} / {results.length} correct!</span>
      }
    </div>
  )}
</div>
```
