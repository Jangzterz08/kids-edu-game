**shuffle function** (copy Fisher-Yates from MatchingGame.jsx):

function shuffle(arr) {

**Component signature:**
```jsx
export default function MemoryMatchGame({ lessons, onComplete })
```

**Card generation from lesson data:**
```jsx
const lesson = lessons[0]; // MemoryMatch uses first lesson's pairs
const cards = useMemo(() => {
  if (!lesson?.pairs) return [];
  return shuffle(
    lesson.pairs.flatMap((pair, i) => [
      { id: `${i}-a`, pairId: i, emoji: pair[0] },
      { id: `${i}-b`, pairId: i, emoji: pair[1] },
    ])
  );
}, []);
```

**State (mirrors MatchingGame pattern):**
- `flipped` (number[]) — indices of currently face-up cards (max 2)
- `matched` (number[]) — pairIds of matched pairs
- `justMatched` (number | null) — pairId briefly set for bounce animation
- `attempts` (number) — total flip-pair attempts
- `locked` (boolean) — blocks taps during 900ms mismatch window

**handleFlip(idx):**
1. If `locked` return (Pitfall 4 prevention)
2. If `flipped.includes(idx)` return (can't re-flip same card)
3. If `matched.includes(cards[idx].pairId)` return (already matched)
4. Build `newFlipped = [...flipped, idx]`
5. `setFlipped(newFlipped)`
6. If `newFlipped.length === 2`:
   - `setLocked(true)`, `setAttempts(a => a + 1)`
   - Compare `cards[newFlipped[0]].pairId === cards[newFlipped[1]].pairId`
   - **Match:** add pairId to matched, set justMatched, clear justMatched after 600ms, `setFlipped([])`, `setLocked(false)`. If all 6 pairs matched: compute `extra = Math.max(0, attempts + 1 - 6)`, `score = Math.max(0, 100 - extra * 8)`, `setTimeout(() => onComplete(score), 700)`
   - **Mismatch:** `setTimeout(() => { setFlipped([]); setLocked(false); }, 900)`

**Helper functions:**
- `isFlipped(idx)`: `flipped.includes(idx) || matched.includes(cards[idx].pairId)`
- `isMatched(idx)`: `matched.includes(cards[idx].pairId)`
- `isJustMatched(idx)`: `justMatched !== null && cards[idx].pairId === justMatched`

**Render (4x3 CSS Grid, centered, max-width 320px):**
```jsx
<div style={styles.container}>
  <h2 style={styles.title}>Match the pairs! 🧠</h2>
  <p style={styles.sub}>Attempts: {attempts}</p>
  <div style={styles.grid}>
    {cards.map((card, i) => (
      <button key={card.id} onClick={() => handleFlip(i)} style={styles.cardOuter}>
        <div style={{
          ...styles.cardInner,
          transform: isFlipped(i) ? 'rotateY(180deg)' : 'rotateY(0deg)',
        }}>
          {/* Front face (card back — "?") */}
          <div style={{
            ...styles.cardFace,
            ...styles.cardFront,
            ...(isMatched(i) ? styles.cardMatched : {}),
            animation: isJustMatched(i) ? 'match-bounce 0.55s ease' : 'none',
          }}>
            {isMatched(i) ? <span style={styles.cardEmoji}>{card.emoji}</span> : <span style={styles.cardQuestion}>?</span>}
          </div>
          {/* Back face (emoji — revealed on flip) */}
          <div style={{
            ...styles.cardFace,
            ...styles.cardBack,
            ...(isMatched(i) ? styles.cardMatched : {}),
          }}>
            <span style={styles.cardEmoji}>{card.emoji}</span>
          </div>
        </div>
      </button>
    ))}
  </div>
</div>
```
